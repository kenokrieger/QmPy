QmPy is a python package containing routines to numerically solve and visualize the schroedinger equation for different potentials.

Documentation¶

  • API-Documentation

Requirements¶

This package requires Python 3.6 or higher and the packages numpy, scipy, matplotlib and os.

Installation¶

For easy installation with pip use:

pip install -i https://test.pypi.org/simple/ qmpy-schrodinger
